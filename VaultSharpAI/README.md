# VaultSharpAI 🧠🛡️

Мощный, безопасный и кроссплатформенный десктопный клиент для работы с искусственным интеллектом, построенный на базе современного стека .NET и Avalonia UI.

Проект создается как универсальный и независимый шлюз к моделям ИИ, а также для глубокого изучения архитектурных паттернов (MVVM, DI), проектирования современных интерфейсов и работы с локальными нейросетями.

## ✨ Ключевые возможности

* **Полная независимость от провайдера:** Подключение к любым OpenAI-совместимым облачным API (DeepSeek, ChatGPT, OpenRouter, кастомные провайдеры и агрегаторы) через гибкую настройку Base URL и API Key.
* **Автоопределение моделей:** Динамическое получение списка доступных на сервере моделей через эндпоинт `/models` в один клик.
* **Продвинутое управление контекстом:** Полная поддержка стриминга ответов (посимвольный вывод текста в реальном времени), переключатели режимов Мышления (Thinking) и глубокого исследования (Deep Research), а также точная настройка лимита токенов.
* **Кроссплатформенность:** Полноценная работа на Windows и Linux благодаря возможностям графического движка Avalonia UI.
